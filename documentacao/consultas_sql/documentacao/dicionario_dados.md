# Dicionário de Dados

Este documento descreve os campos utilizados na base fictícia do projeto.

| Campo | Tipo | Descrição |
|-------|------|-----------|
| id_cliente | Inteiro | Identificador único do cliente |
| nome_cliente | Texto | Nome completo do cliente |
| data_criacao | Data | Data de criação do registro |
| status | Texto | Situação atual do cliente (Ativo/Inativo) |

## Observações

- Os dados são fictícios para fins de estudo.
- O dicionário ajuda na padronização e entendimento da base.
